# Personal Portfolio

A minimalist personal portfolio website built with Next.js and Tailwind CSS.

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first CSS
- **Lucide React** - Icon library

## Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Features

- Clean, minimalist design with generous white space
- Sticky navigation with smooth scrolling
- Responsive mobile-first design
- Animated scroll-down indicator on hero section
- Single-page scrolling architecture

## Sections

- **Home** - Hero with name, tagline, and hook line
- **About** - Background and introduction
- **Projects** - Showcase of work
- **Experience** - Career history
- **Skills** - Technical and professional skills
- **Contact** - Get in touch
